# DSO Dashboard — GitHub Pages

Live DSO dashboard for your team. Pulls from Moneybird, hosted free on GitHub Pages.

**Your team's URL:** `https://YOUR-USERNAME.github.io/dso-dashboard/`

---

## Setup (15 minutes total)

### Step 1 — Create a GitHub account & repository (5 min)

1. Sign up free at **github.com** if you don't have an account
2. Click **New repository**
3. Name it: `dso-dashboard`
4. Set to **Public** (required for free GitHub Pages)
5. Click **Create repository**

---

### Step 2 — Upload this folder to GitHub (3 min)

**Option A — GitHub Desktop (easiest, no terminal needed):**
1. Download **GitHub Desktop** from desktop.github.com
2. Clone your new repo: File → Clone Repository → `dso-dashboard`
3. Copy all files from this folder into the cloned folder
4. In GitHub Desktop: write commit message "Initial setup" → click **Commit** → **Push**

**Option B — Terminal:**
```bash
cd path/to/dso_github
git init
git remote add origin https://github.com/YOUR-USERNAME/dso-dashboard.git
git add .
git commit -m "Initial setup"
git push -u origin main
```

---

### Step 3 — Enable GitHub Pages (2 min)

1. Go to your repo on github.com → **Settings** → **Pages** (left sidebar)
2. Under **Source**: select **Deploy from a branch**
3. Branch: **main** / Folder: **/docs**
4. Click **Save**
5. Wait ~2 minutes → your dashboard is live at:
   `https://YOUR-USERNAME.github.io/dso-dashboard/`

---

### Step 4 — Configure & run the sync script (3 min)

Install dependencies:
```bash
pip install requests
```

Set your credentials:
```bash
# Mac/Linux
export MONEYBIRD_TOKEN="your_token_here"
export MONEYBIRD_ADMIN_ID="123456789"

# Windows
set MONEYBIRD_TOKEN=your_token_here
set MONEYBIRD_ADMIN_ID=123456789
```

Run the first sync:
```bash
python generate_dashboard.py --push
```

This fetches your Moneybird data, writes `docs/data.json`, and pushes to GitHub.
Your dashboard updates live within seconds.

---

### Step 5 — Schedule automatic updates

**Mac/Linux — cron job (runs every weekday at 08:00):**
```bash
crontab -e
# Add this line:
0 8 * * 1-5 cd /path/to/dso_github && python generate_dashboard.py --push >> dso.log 2>&1
```

**Keep it running in the background:**
```bash
python generate_dashboard.py --push --loop 1   # refresh every hour
```

**Windows — Task Scheduler:**
1. Search "Task Scheduler" → Create Basic Task
2. Trigger: Daily at 08:00
3. Action: `python C:\path\to\generate_dashboard.py --push`

## Updating credentials safely

Never put your API token in a file you push to GitHub.
Always use environment variables or a `.env` file (add `.env` to `.gitignore`).

```bash
# .env file (never commit this)
MONEYBIRD_TOKEN=your_token_here
MONEYBIRD_ADMIN_ID=123456789
```

```bash
# Load it before running
source .env && python generate_dashboard.py --push
```
